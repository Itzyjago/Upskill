# Upskill

A personal learning hub — notes, roadmaps, and reference material I build while
leveling up across software engineering and DevOps.

## Why this repo exists

Learning sticks when it's written down. This repo is my second brain for the
skills I'm actively practicing: short, practical notes I can grep later instead
of re-Googling the same thing every few weeks.

## Structure

| Path | What's inside |
|------|---------------|
| `ROADMAP.md` | The skills I'm working through and where I am with each |
| `notes/` | Topic notes (JavaScript, Python, Git, Docker, SQL, ...) |
| `resources/` | Curated links worth keeping |
| `PROGRESS.md` | A running log of what I learned and when |

## How I use it

1. Pick a topic from the roadmap.
2. Work through it hands-on.
3. Write down the parts that surprised me into `notes/`.
4. Log the session in `PROGRESS.md`.
